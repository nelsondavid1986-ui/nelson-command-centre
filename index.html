Android Use

Phone Reminder Setup

  1. Open this app in Chrome on Android.
  2. Tap the Chrome three-dot menu.
  3. Tap Add to Home screen.
  4. Use each task’s Google Calendar button to create phone reminders.

This version does not directly control the Android Clock alarm. It uses Google Calendar reminders, which are safer and more reliable for the first version.

Local-first app. Data is stored in this browser using LocalStorage.
