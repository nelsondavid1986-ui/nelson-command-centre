V2.6.5 Final Notes

Validated stable build

Graphs not shown

Graphs now draw after every render, show fallback messages, and no longer depend on broken old chart logic.

Dropdowns empty

Food, task category, coach mode, templates and body selectors are populated during startup and every render.

Buttons not working

Buttons have explicit event bindings with safety checks. Broken missing-ID crashes are avoided.

Body target clarity

Recommended baseline is built around your 172cm profile, 68kg default, and ~20% body fat starting estimate. You can edit all values.

Security warning

Do not paste OpenAI API keys into public GitHub Pages code. This app uses safe copy/paste coach prompt workflow.
