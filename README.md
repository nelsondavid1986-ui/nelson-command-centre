# Nelson Command Centre

A mobile-first personal task, countdown, Google Calendar and daily review web app.

## How to use on Android

1. Download and unzip the folder.
2. Open `index.html` in Chrome.
3. Tap the Chrome three-dot menu.
4. Tap **Add to Home screen**.
5. Use the **Google Calendar** button on each task to create a calendar reminder.

## Features

- Add / edit / delete tasks
- Project categories
- Priority and status system
- Deadline countdown warnings
- Today’s Top 3
- Project progress tracker
- Daily review log
- Google Calendar event link
- LocalStorage saving
- Export / import backup JSON

## Limitation

This first version does not directly control the Android Clock alarm. It uses Google Calendar reminders instead.
